# Reddit
A reddit clone written using node.js / express.js / mongodb / passport.js

Work in Progress!
![Image](https://i.imgur.com/bxyvQj0.png)

### Current Working:
* View subreddits
* View posts
* Submit posts/links/comments
* Create subreddits
* Saving / deleting posts and comments
* Upvote / downvote posts and comments
* Relative time
* Database
* Login
* Authentication
